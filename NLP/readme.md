# 自然语言处理相关

- nlp_bert_small.ipynb 使用bert small模型二分类（Fake news）
- nlp_bert_cls.ipynb 使用bert模型二分类（Fake news）
- nlp_cls_bertandRoberta.ipynb 同上 模型进行了修改
- nlp_cls_nltk.ipynb 使用nltk 二分类
